# Backend Node.js/Express

## Cấu trúc thư mục
- controllers/: Xử lý logic cho từng API
- services/: Xử lý nghiệp vụ/phức tạp
- routes/: Định nghĩa các endpoint
- models/: Định nghĩa schema/data
- middlewares/: Middleware dùng chung
- utils/: Hàm tiện ích
- config/: Cấu hình
- app.js: Khởi tạo app Express
- server.js: Chạy server

## Khởi động
```bash
npm install
npm run dev
```
